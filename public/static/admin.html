CSV Format Instructions

Your CSV file should have the following columns (header row required):

  • name - Full name (required)
  • email - Email address (required, must be unique)
  • role - User role (optional, defaults to "consultant")
  • employee_id or eeid - Employee ID (optional)
  • join_date - Join date in YYYY-MM-DD format (optional)
  • manager_id or boss_id - Manager's user ID (optional)
  • division - Division/Department (optional)
  • region - Region (optional)
  • location - Office location (optional)
  • title - Job title (optional)
